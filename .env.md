Environment Variables Configuration

Create a `.env` file inside the agent directory.

### 📌 Required Variables

| Variable                  | Description                                    |
| ------------------------- | ---------------------------------------------- |
| GOOGLE_GENAI_USE_VERTEXAI | Set to TRUE to enable Vertex AI authentication |
| GOOGLE_CLOUD_PROJECT      | Your Google Cloud Project ID                   |
| GOOGLE_CLOUD_LOCATION     | Regional endpoint (e.g., global)               |
| MODEL                     | LLM version (e.g., gemini-3-flash-preview)     |

### 📝 Example `.env`

```env
GOOGLE_GENAI_USE_VERTEXAI=TRUE
GOOGLE_CLOUD_PROJECT=your-project-id
GOOGLE_CLOUD_LOCATION=global
MODEL=gemini-3-flash-preview
```

## 📌 Notes

* The `.env` file is mandatory for authentication.
* Prefer using virtual environments (`venv`).
* Ensure your Google Cloud project has Vertex AI enabled.
